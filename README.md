# 🎬 Netflix Power BI Dashboard

## 📌 Project Overview
This project is an interactive **Netflix Power BI Dashboard** created to analyze Netflix's content library. The dashboard provides insights into movies, TV shows, genres, ratings, release trends, directors, and global content distribution through interactive visualizations.

---

## 📊 Dashboard Features

- 📈 Total Netflix Titles
- 🎥 Total Movies
- 📺 Total TV Shows
- 🌍 Country-wise Content Distribution (Filled Map)
- 🎭 Top 10 Genres
- ⭐ Content Ratings Analysis
- 🎬 Top Directors
- 📅 Release Year Trend
- 🍩 Movies vs TV Shows Distribution
- 🎛 Interactive Filters and Visualizations

---

## 🛠 Tools & Technologies

- Microsoft Power BI
- Power Query
- DAX
- Microsoft Bing Maps
- Netflix Dataset

## 📈 Key Insights

- Movies make up the majority of Netflix's content.
- Drama and International Movies are among the most popular genres.
- The number of Netflix releases increased significantly after 2015.
- The United States contributes the highest number of Netflix titles.
- TV Shows represent a smaller portion compared to Movies.
